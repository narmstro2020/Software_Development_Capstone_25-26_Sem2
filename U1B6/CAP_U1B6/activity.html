ğŸ“š Quick Reference

Writing Files

File.WriteAllText("file.txt", "content"); // Overwrite File.WriteAllLines("file.txt", stringArray); // Overwrite File.AppendAllText("file.txt", "new text"); // Add to end

Reading Files

string text = File.ReadAllText("file.txt"); // One string string[] lines = File.ReadAllLines("file.txt"); // Array of lines bool exists = File.Exists("file.txt"); // Check first!

StreamWriter (using block)

using (StreamWriter writer = new StreamWriter("file.txt")) { writer.WriteLine("line of text"); } // To append: new StreamWriter("file.txt", true)

File Operations

File.Copy("source.txt", "dest.txt", true); // Copy file File.Delete("file.txt"); // Delete file

Date/Time Formatting

string stamp = DateTime.Now.ToString("yyyy-MM-dd HH:mm"); // Result: "2025-01-15 14:30"
